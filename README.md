# All Aboard Bootcamp Final Project

This is my app for the final project for the All Aboard Bootcamp by John Crepezzi


## Run Locally

Clone the project

```bash
  git clone https://github.com/All-Aboard-Bootcamp/dotRamsey-final.git
```

Go to the project directory

```bash
  cd dotRamsey-final
```

Install dependencies

```bash
  bundle install && npm install
```

Create databases make sure you have PostgreSQL
```bash
rails db:create
```

Migrate
```bash
rails db:migrate
```

Start the server

```bash
  rails s
```
